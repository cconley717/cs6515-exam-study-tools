DFS: Table format showing pre-order and post-order numbers of the traversal. Directed or undirected graphs supported.

BFS: Table format showing distances from a source vertex to all other vertices for each iteration. Directed or undirected graphs supported.

Dijkstra: Table format showing weighted shortest distances from a source vertex to all other vertices for each iteration.

Bellman-Ford: Table format showing weighted shortest distances from a source vertex to all other vertices for each iteration.

Floyd-Warshall: Table format showing weighted shortest distances for all pairs of vertices for each iteration.

Prim: Table format showing the minimum spanning tree for a weighted graph.

Kruskal: Table format showing the minimum spanning tree for a weighted graph.

Max Flow: Table format showing the original graph and another table format showing the residual graph.

SCC: Outputs the SCCs of a directed graph. Each SCC is a string and is printed on its own line.

Edges: Table format showing starting traversals for all vertices and the types of edges found.
